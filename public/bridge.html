2. Create iOS Shortcut (once)

This generates the JavaScript code for your Shortcut. Tap "Copy Code", then follow the steps below.

  1. Open the Shortcuts app
  2. Tap + to create a new shortcut
  3. Tap Add Action, search for "Run JavaScript on Web Page"
  4. Tap the default code and Select All → Paste (the code you just copied)
  5. Tap the shortcut name at top, rename to "Milk Bridge"
  6. Tap the (i) button → enable "Show in Share Sheet"
  7. Under "Share Sheet Types" keep only Safari web pages
  8. Tap Done
3. Use it
  1. Open mylactalis.com.au in Safari and log in
  2. Tap the Share button (box with arrow)
  3. Scroll down and tap "Milk Bridge"
  4. The floating widget appears — it auto-syncs the schedule
