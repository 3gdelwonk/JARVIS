Milk Manager Bridge

Sync schedules & submit orders from your phone

1. Configure once
2. Test Shortcuts once

First let's confirm iOS Shortcuts can run JavaScript on the Lactalis page. This takes 2 minutes.

  1. Open Shortcuts app
  2. Tap + → Add Action
  3. Search "Run JavaScript on Web Page" and add it
  4. Tap the code area, Select All → Paste
  5. Add a second action: search for "Show Alert" and add it
  6. In the Show Alert box, tap "Result" placeholder (it should already say "Shortcut Input" — tap it and change to "Run JavaScript on Web Page")
  7. Tap shortcut name → rename to "Bridge Test"
  8. Tap Done
  9. Open mylactalis.com.au in Safari
  10. Tap Share (box with arrow) → scroll down → tap "Bridge Test"
You should see an alert popup showing the page title and cookie status. If nothing happens, Shortcuts may not work on your iOS version — skip to Step 4 (Alternative).
3. Create Sync Shortcut once

If the test worked, create the real shortcut. This one syncs the schedule and checks for pending orders — no widget needed, just tap and go.

  1. Open Shortcuts app
  2. Tap + → Add Action → "Run JavaScript on Web Page"
  3. Tap the code, Select All → Paste
  4. Add second action: "Show Alert" (shows results)
  5. In Show Alert, set input to "Run JavaScript on Web Page" result
  6. Rename to "Milk Bridge"
  7. Tap (i) → enable "Show in Share Sheet"
  8. Tap Done

To use: Open Lactalis in Safari → Share → Milk Bridge. An alert shows the sync result.

4. Alternative: Userscripts App if shortcuts fail

If Shortcuts doesn't work, install the free Userscripts Safari extension from the App Store. It reliably injects JavaScript into any page.

  1. Install Userscripts from the App Store
  2. Open Settings → Safari → Extensions → Userscripts → enable it
  3. Allow it for mylactalis.com.au
  4. Open Userscripts app, tap + to create a new script
  5. Set @match to https://*.lactalis.com.au/*
  6. Tap "Copy Userscript Code" below and paste it in
  7. Save — the bridge now runs automatically on every Lactalis page load
